## Installation
#### Go to web site link and use freely. Optionally download files to git clone run npm i, npm i express, npm i fs, npm uuid, npm i nodemon, and npm i path. Then for local use run nodemon server.js and open browser to http://localhost:3001. Optionally use Heroku to create your own Heroku copy. Git, add/commit/push to your own repo then create heroku cmd then git push heroku. Once it completes use heroku open to open your own unique note taking web browser with https security. 

## Usage
#### Take notes and posted them to the Web, stored and persist on page load and change. Items can be deleted on completeion. Use link: https://notus12potus35.herokuapp.com/
